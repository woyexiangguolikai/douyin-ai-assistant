# 产品需求文档 PRD

## 1. 项目概述

**项目名称**：抖音AI直播助手
**目标用户**：抖音娱乐主播（聊天、情感、灵异故事、才艺等类型）
**核心场景**：直播间实时弹幕采集 + AI 智能提词，帮助主播解决“不知道回什么”“冷场”“故事素材不够”等痛点
**版本**：v1.0.0

## 2. 用户场景

### P0 核心场景
1. **弹幕实时采集**：连接直播间后实时抓取观众弹幕，支持多种采集方案
2. **AI 智能回复**：根据弹幕内容、主播人设自动生成回复话术和故事素材
3. **打字机逐字显示**：生成的回复逐字展示，主播可边看边用
4. **左右分栏视图**：左侧弹幕列表，右侧 AI 回复，中间虚线连接对应关系

### P1 重要场景
5. **人设配置**：支持多套主播人设模板（默认、电台、才艺等），自定义性格、语气、禁忌话题
6. **过滤规则**：多种过滤策略（最短字数、纯表情、重复刷屏、广告关键词等）
7. **历史导出**：AI 回复记录可导出为 TXT 文件，便于每周复盘

### P2 增强场景
8. **管理后台**：多角色权限管理、直播间统一配置、状态监控
9. **悬浮窗模式**：窗口置顶、可调透明度，适合直播时屏幕空间有限的场景

## 4. 界面规范

检视“彩玉（Obsidian）脚本”多弘颜色主题，主色为 #ccff00（银晶绿），背景为纯黑（#000000）和观音石表面（#0c0c0c）。全部卡片采用玻璃态设计：backdrop-filter: blur(16px)，边框 1px solid rgba(255,255,255,0.1)。
